# 구조

* #### 프롬프트

구문 분석기 -> 토큰

## 기능

* 프롬프트
  * `ctrl-C`
  * `ctrl-D`
  * `ctrl-\`
* 구문 분석 우선순위
  * `'`
  * `"`
    * `$VAR`
* 메타문자
  * `>`
  * `<`
  * `<<`
  * `>>`
  * `|`
* 프로세스 실행 및 종료
  * `$?`
* 빌트인 함수
  * echo (-n)
  * cd
  * pwd
  * export
  * unset
  * env
  * exit
